# MangLua definitions generator

MangLua LUA definition files generation utility

#### [Download](https://github.com/Sillerstone/manglua-definitions-generator/releases/latest)

## Usage

### Windows

Command Prompt

```sh
mlua-gen help
```

PowerShell

```sh
.\mlua-gen help
```

### Linux & macOS

```sh
chmod +x ./mlua-gen
./mlua-gen help
```

## Building

```sh
cargo build --profile release
```
